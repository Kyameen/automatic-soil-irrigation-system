# Automatic Soil Irrigation System

This project implements an Arduino-based automatic irrigation system that:
- Monitors soil moisture and rain
- Controls a 12V water pump using a relay
- Measures temperature and humidity using DHT11
- Sends real-time data to a mobile phone via HC-05 Bluetooth

## Hardware Used
- Arduino Uno
- Soil Moisture Sensor
- Rain Sensor
- Relay Module
- 12V Water Pump
- DHT11
- HC-05 Bluetooth Module

## Features
- Automatic pump control
- Water waste reduction
- Wireless monitoring
- Safe non-blocking timing using millis()
